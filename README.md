# LifeSync

LifeSync is a personal productivity and management application built with React, Firebase, Capacitor, and Android.

## Features

- User Authentication
- Dashboard
- Expense Tracker
- Goals Tracker
- Study Planner
- Bazaar Management
- Android APK Support

## Technology Stack

- React
- Vite
- Firebase
- Capacitor
- Android Studio

## Development

### Install dependencies

```bash
npm install
```

### Run locally

```bash
npm run dev
```

### Build production version

```bash
npm run build
```

### Sync Android project

```bash
npx cap sync android
```
